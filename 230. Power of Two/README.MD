### java

#### 方法一 

如果一个数用2求模之后为0，则将这个数向右移动一位（除2）

最后返回结果对比这个数最后是否等于1

**核心代码**
```java
if (n<=0)
	return false;
int mod = n%2;  //如果不是2的倍数，不能进入while循环，
while (mod == 0 && n>0){
	n = n>>1;
	mod = n%2;
}
return n==1;
```java


#### 方法2

如果一个数是2的幂次，则说明这个数只有二进制中只包含了一个1，采用 n &（n-1） 即可判断出来是否是2的幂次

**核心代码**
```java
if (n<=0) return false;
return (n&(n-1))==0;
```
